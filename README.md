## Setup Steps

### 1. Install dependencies
```bash
pip install scikit-learn joblib pandas openpyxl
```

### 2. Train the model (run once)
```bash
# Place Interior_Planning_Dataset_10000.xlsx in your project root
# Edit APP_NAME in train_and_save_model.py to match your app folder
python train_and_save_model.py
```
This creates `<your_app>/ml_assets/cost_model.pkl` and `encoders.pkl`.

### 3. Add Gemini API Key to settings.py
```python
# settings.py
GEMINI_API_KEY = "AIzaSy..."   # Get from https://aistudio.google.com/
```

### 4. File placement
```
your_project/
├── your_app/
│   ├── ml_assets/
│   │   ├── cost_model.pkl       ← generated by train script
│   │   ├── encoders.pkl         ← generated by train script
│   │   └── model_mapping.json   ← generated by train script
│   ├── templates/
│   │   ├── base.html
│   │   ├── home.html            ← upgraded
│   │   ├── result.html          ← upgraded (shows blueprint)
│   │   ├── profile.html         ← upgraded
│   │   ├── login.html           (unchanged)
│   │   └── register.html        (unchanged)
│   ├── ml_model.py              ← upgraded
│   ├── views.py                 ← upgraded
│   └── urls.py                  ← upgraded (added /api/estimate/, /logout/)
└── train_and_save_model.py
```

### 5. Update urls.py (project-level)
Make sure your project urls.py includes logout and api routes — these are already in the app-level urls.py.

### 6. Run migrations & start
```bash
python manage.py migrate
python manage.py runserver
```

## Gemini API Key (Free)
1. Go to https://aistudio.google.com/
2. Click **Get API Key** → **Create API key**
3. Copy the key (starts with `AIza…`)
4. Add to `settings.py`: `GEMINI_API_KEY = "AIza…"`

Blueprint generation uses `gemini-2.0-flash-exp-image-generation` model.
